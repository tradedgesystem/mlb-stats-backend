Test Instructions

This test will verify that your extension correctly saves and restores:

  • Saved players
  • Selected statistics
  • Year selection
  • Mode selection (hitters/pitchers)
  • Date range settings

Step 1: Initial Setup

Actions to perform:
  1. Open the MLB Stats extension popup
  2. Verify the browser console is open (F12 or Cmd+Option+I)
  3. Look for [Init] messages in the console
  4. The extension should load with default settings

Step 2: Add Players

Actions to perform:
  1. Make sure Mode is set to "Hitters"

  5. Search for "Freddie Freeman" and click "Add"

Troubleshooting

If persistence is not working:
  1. Check the browser console for [Persistence] errors
  2. Verify storage permissions are in manifest.json
  3. Reload the extension in chrome://extensions
  4. Click "Clear Chrome Storage" above and restart the test
  5. Look for console errors during save/load operations
